# 上海历史研究系统 Web 应用

基于 React + FastAPI 的上海历史保护建筑和历史地图展示系统。

## 功能模块

### 1. 历史保护建筑模块
- 地图展示所有历史保护建筑位置
- 按保护等级和文物批次筛选
- 点击建筑查看详细信息和历史照片
- 图片放大查看，支持键盘左右键切换

### 2. 历史地图模块
- 历史地图检索与筛选
- 支持多条件筛选
- 点击查看地图大图和详细信息
- 图片放大查看，支持键盘左右键切换

## 技术栈

- 前端: React 18 + TypeScript + Vite + TailwindCSS + Leaflet
- 后端: FastAPI + SQLAlchemy + MySQL
- 部署: Docker
