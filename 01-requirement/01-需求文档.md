## 3. 用户流程（User Flow）

### 3.1 历史保护建筑模块（Map）
1. 用户进入系统，默认显示历史保护建筑地图页面
2. 用户可以在地图上看到所有建筑标记点
3. 用户可通过下拉菜单筛选保护等级或文物批次
4. 用户点击建筑标记，弹出详情弹窗
5. 弹窗显示顺序：
   - 首先显示主图（IMAGE_PATH）
   - 其次显示历史照片（PHOTOGRAPHY表）
   - 最后显示建筑详细信息
6. 用户点击任意图片放大查看，支持键盘左右键循环切换图片

### 3.2 历史地图模块（Maps）
1. 用户点击左侧菜单切换到历史地图模块
2. 用户看到历史地图列表，支持多条件筛选
3. 用户点击地图条目，弹出详情弹窗
4. 用户可点击图片放大查看，支持键盘左右键切换图片

### 3.3 历史建筑管理模块（Buildings）
1. 用户点击左侧菜单切换到建筑管理模块
2. 用户看到历史建筑列表，支持多条件筛选
3. 用户可点击"添加建筑"按钮新增记录
4. 用户可点击建筑行的"编辑"按钮修改信息
5. 用户可点击"删除"按钮删除建筑（含确认对话框）
6. 用户点击建筑行进入详情，查看完整信息和关联照片

### 3.4 历史照片管理模块（Photos）
1. 用户点击左侧菜单切换到照片管理模块
2. 用户看到历史照片列表，支持多条件筛选
3. 用户点击照片进入详情，查看完整信息
4. 用户可点击照片勾选进入多选模式
5. 在多选模式下选择照片，点击"绑定到建筑"按钮
6. 在详情弹窗中可绑定/解除建筑，添加/删除标签和类型

## 4. 图片灯箱要求

- 所有图片（主图+历史照片）合并为一个数组
- 按索引顺序排列：[主图, 照片1, 照片2, ..., 照片N]
- 按←键：index = (index - 1 + 总数) % 总数（循环）
- 按→键：index = (index + 1) % 总数（循环）
- 显示计数器：当前索引+1 / 总数
- ESC 关闭灯箱
