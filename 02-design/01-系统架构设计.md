## 3. 前端组件架构

| 路由 | 组件 | 描述 |

## 4. 界面交互规范

### 4.1 弹窗层级

| 层级 | z-index | 用途 |
|------|---------|------|
| Leaflet MapPane | 400-700 | 地图内部元素 |
| Leaflet ControlPane | 800 | 地图控件 |
| Modal Backdrop | 999 | 遮罩层 |
| Modal Container | 1000 | 弹窗主体 |
| Image Lightbox | 3000 | 图片灯箱 |

### 4.2 键盘交互

| 键 | 场景 | 行为 |
|----|------|------|
| ESC | 弹窗打开时 | 关闭弹窗 |
| ESC | 灯箱打开时 | 关闭灯箱 |
| ← | 详情弹窗（地图） | 切换到上一个地图 |
| → | 详情弹窗（地图） | 切换到下一个地图 |
| ← | 图片灯箱 | 显示上一张图片（循环） |
| → | 图片灯箱 | 显示下一张图片（循环） |

### 4.3 颜色规范（PhotosPage）

| 元素 | 颜色 |
|------|------|
| 标签（Tags） | bg-purple-100 text-purple-700 |
| 类型（Types） | bg-orange-100 text-orange-700 |
| 建筑（Buildings） | bg-amber-100 text-amber-700 |
