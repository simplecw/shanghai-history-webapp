## 5. 表关系图

### 5.1 保护建筑照片关联链路

```
SHANGHAI_HISTORIC_PRESERVATION_BUILDING (保护建筑表，579条)
       │
       │ 1:1 (通过 ORIGINAL_NAME 匹配)
       ▼
Shanghai_history_building (历史建筑表，1804条)
       │
       │ 1:N
       ▼
SHANGHAI_HISTORY_BUILDING_PHOTOGRAPHY (关联表，779条)
       │
       │ N:1
       ▼
SHANGHAI_HISTORY_PHOTOGRAPHY (照片表，6624条)
```

### 5.2 建筑类型关联

```
Shanghai_history_building (历史建筑表)
       │
       │ 1:N
       ▼
Shanghai_history_building_type (建筑类型关联表)
       │
       │ N:1
       ▼
CODE_LOOKUP (代码表，CODE_TYPE="建筑类型")
```

### 5.3 照片类型/标签关联

```
SHANGHAI_HISTORY_PHOTOGRAPHY (照片表)
       │

       │              │
       │              └──N:1── CODE_LOOKUP (CODE_TYPE="照片类型")
       │
       └──1:N── SHANGHAI_HISTORY_PHOTOGRAPHY_tag (照片标签关联)
```

### 5.4 保护建筑-历史建筑关联

SHANGHAI_HISTORIC_PRESERVATION_BUILDING (保护建筑表)

## 7. 坐标系统

### 7.1 保护建筑表坐标

数据库中存储的坐标是百度 BD-09 坐标系，需要转换为 WGS-84（Leaflet 地图使用）。

转换公式（BD-09 → GCJ-02 → WGS-84）：
```
lat_wgs = lat_bd - 0.006 + adjustment
lng_wgs = lng_bd - 0.0065 + adjustment
```

### 7.2 历史建筑表坐标

历史建筑表使用 X_AXIS / Y_AXIS 字段存储坐标。

### 7.3 上海范围验证

- 纬度：30.5°N ~ 32°N
- 经度：120.5°E ~ 122.5°E
